 硬件配置：
 ---3.5寸液晶屏引脚使用说明---
        
           CS2  -->  PA2
           CS1  -->  PA1
           RES  -->  PA0
           DC   -->  PA4
           BLK  -->  PA3
           CLK  -->  PA5
           MOSI -->  PA7
           MISO -->  PA6
             
 ---按键引脚使用说明---
            
          K0  -->  PB10
          K1  -->  PB11
          K2  -->  PB12
          K3  -->  PB13
            
 ---串口通讯引脚使用说明--- 
               （串口1）       
         RXD  -->  PA9(TXD)
         TXD  -->  PA10(RXD)
            
 ---外部中断引脚使用说明---
            
        激光接收端 --> PA8
		
		
软件注释：

MenuFlag = 0 :表示主界面（用于选则工作模式：单圈计时模式，多从机工作模式）
MenuFlag = 1 :表示单圈测试界面（单圈计数界面标志）
MenuFlag = 2 :表示从机通讯匹配界面（用于连接和记录从机数量）
MenuFlag = 3 :表示多从机测试界面（用于直线加速，区间测速，滑行实验等）
MenuFlag = 4 :表示单圈测试数据显示界面
MenuFlag = 5 :表示多机测试数据显示界面
MenuFlag = 6 :表示单圈测试数据清空界面
MenuFlag = 7 :表示多从机测试数据清空界面
